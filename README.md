# Story Game

A simple web-based interactive story game styled after titles like *Written Realms*.

## Features

- Branching narrative defined in JavaScript.
- Clean, dark-themed interface with choices rendered as buttons.
- Easy to extend by editing the `story` object in `script.js`.

## Getting Started

1. Open `index.html` in a browser (double-click or serve using a static file server).
2. Read the scene text and click on choices to progress through the story.
3. Modify or expand the story by editing `script.js`.

## Structure

- `index.html`: Contains the game container and links to the CSS/JS.
- `style.css`: Simple styling for the game UI.
- `script.js`: Core logic and story data.

Feel free to build your own branches, add variables like health or inventory, or hook up progress save/load logic.
